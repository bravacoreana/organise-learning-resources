# AI Resource Roadmap

Next.js 기반 학습 워크스페이스입니다.

## 기능

- 링크 / 파일 추가 후 자동 분류
- 로드맵, 카테고리, 난이도별 보기
- 엑셀 느낌의 표 보기
- 학습 진도 필터링
- 리소스 상세 정보, 요약, 학습 목표, 준비도, 분석 메모
- 리소스별 노트 작성
- 노트만 모아보기
- JSON 내보내기 / 불러오기

## 실행

```bash
cd "/Users/liha/Desktop/Vibe Codes/ai-resource-roadmap"
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 을 열면 됩니다.
